DMMT Examples

Wikipedia - Clarity in Navigation and Information Structure

Rule Demonstrated: Don't Make Me Think

Description: Wikipedia exemplifies the principle of clear navigation and information structuring, making it effortless for users to find and comprehend the content. The search bar is prominently placed, categories are well-organized, and internal links guide users to related topics seamlessly, embodying the "Don't Make Me Think" ethos by facilitating an intuitive user experience.

Landbook - Aesthetic and Minimalistic Design

Rule Demonstrated: Keep it simple

Description: Landbook showcases a minimalist design that emphasizes simplicity, engaging users with its striking use of colors and straightforward navigation. This design approach reduces cognitive load, making it easier for users to interact with the site and appreciate its offerings without feeling overwhelmed.

SaaS Landing Page - Mobile-First Approach

Rule Demonstrated: Make it obvious what’s clickable

Description: SaaS Landing Page excels in mobile design, clearly distinguishing clickable elements, ensuring that navigation is intuitive on smaller screens. This design principle enhances the mobile user experience by providing clear, tap-friendly buttons and links that facilitate easy navigation and interaction.

Websites Demonstrating Design Principle Violations

Daily Mail - Excessive Advertisements

Rule Violated: Minimize the noise

Description: Daily Mail’s website suffers from an overabundance of advertisements, which clutter the layout and distract from the content. This excessive noise complicates the user experience, making it challenging to focus on the actual news articles amidst the sea of promotional content.

New York University (NYU) - Poor Color Contrast and Layout

Rule Violated: Create a clear visual hierarchy

Description: NYU’s website struggles with visual hierarchy due to its overuse of similar shades of purple and a confusing grid layout. This lack of contrast and unclear section divisions make it difficult for users to navigate and distinguish between different parts of the site, leading to a confusing user experience.

MyUS.com - Poor Use of Space and Inconsistent Typography

Rules Violated: Keep it simple and Create a clear visual hierarchy

Description: MyUS.com encounters significant issues with its use of whitespace and typography, leading to a cluttered and visually unappealing layout. The website packs an abundance of information into limited space without proper segmentation, resulting in a disorganized appearance. Furthermore, the inconsistent use of font sizes and styles throughout the site adds to the confusion, detracting from the user's ability to easily digest the content. Proper use of heading tags and more structured layout could greatly enhance readability and improve the overall user experience.
